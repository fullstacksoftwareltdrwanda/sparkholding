# Spark Holding Group - Money Lending Group

A comprehensive financial management system for Spark Holding Group, a leading money lending group operating in Rwanda.

## Features
- Money Lending Services
- Strategic Investment Tracking
- Financial Advisory
- Investor Dashboard & Portfolio Management
- Audit Logs & Secure Transactions

## Tech Stack
- Next.js 14
- Prisma (ORM)
- Lucide React (Icons)
- Tailwind CSS
- React Hot Toast

## Development

```bash
npm install
npm run dev
```
